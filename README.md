### Portfolio Website Template
This repository contains the source code for my personal portfolio website, built with **Angular (standalone components)** and **PrimeNG**. It features dark/light mode, fully secured headers, a GDPR-compliant glassmorphism cookie consent manager, and build-time font optimizations.

#### Setup & Build
1. Install dependencies:
   ```bash
   npm install
   ```
2. Build for production:
   ```bash
   npm run build
   ```
3. Run unit tests:
   ```bash
   npm run test:coverage
   ```
4. Run development environment:
   ```bash
   ng serve
   ```

For detailed Firebase Actions and Github Pages setup, see [GITHUB_ACTIONS_SETUP.md](./GITHUB_ACTIONS_SETUP.md) and [GITHUB_PAGES_MIGRATION.md](./GITHUB_PAGES_MIGRATION.md).
